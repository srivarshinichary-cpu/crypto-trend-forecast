Forecasting Cryptocurrency Trends Using Deep Learning

 Project Overview

This project focuses on forecasting Bitcoin cryptocurrency trends using Deep Learning techniques, specifically Long Short-Term Memory (LSTM) networks. The model analyzes historical Bitcoin price data and predicts future closing prices based on time-series patterns.

The project includes:

Data preprocessing

Exploratory Data Analysis (EDA)

Feature engineering

LSTM model training

Future price forecasting

Visualization of prediction results



---

 Features

Bitcoin price prediction using LSTM

Time-series forecasting

Data normalization using MinMaxScaler

Moving average & volatility analysis

Future 30-day forecasting

Visualization of actual vs predicted prices

 Model Used

The project uses an LSTM (Long Short-Term Memory) neural network for time-series forecasting.

Model configuration:

LSTM Layer: 10 neurons

Time Step: 15 days

Epochs: 200

Batch Size: 32

Optimizer: Adam

Loss Function: Mean Squared Error (MSE)
